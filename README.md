# 🗣️ Dialekto

Dialekto is a mobile-first web application prototype designed to help users translate and communicate across Philippine dialects. It combines text translation, voice interaction, phrase libraries, cultural insights, and pronunciation practice into a single user-friendly interface.

## ✨ Features

### 🌐 Dialect Translation
- Translate between major Philippine dialects:
  - Tagalog
  - Cebuano
  - Ilocano
  - Waray
  - Kapampangan
  - Hiligaynon
- Language swap functionality
- Translation output panel
- Audio playback support

### 🎙️ Voice Translation
- Microphone-enabled voice input
- Voice-to-text interaction
- Pronunciation playback

### 📚 Phrase Library
- Organized phrase categories:
  - Greetings
  - Travel
  - Market
  - Emergency
- Bookmark favorite phrases
- Search functionality
- Custom phrase groups

### 🚨 Urgent Communication Kit
Quick-access emergency and help phrases for critical situations.

### 🌴 Cultural Tips
Regional cultural information and etiquette tips to improve communication and understanding.

### 💡 Smart Suggestions
Context-aware phrase recommendations for:
- Travel
- Shopping
- Daily conversations

### 🔊 Pronunciation Practice
Practice commonly used words and phrases with audio support.

### ⚙️ User Settings
- Primary dialect selection
- Downloadable language packs
- Code-switching mode
- Voice settings
- Cultural tips preferences

